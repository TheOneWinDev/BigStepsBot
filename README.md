# BigStepsBot

Реализовано
1. Список пользователей, которым доступен бот
2. Обработка базы mySQL
3. Показ погоды командой /weather
4. Пересылка постов из основного канала

В планах:
1. Логгирование команд пользователя
2. Меню выбора действий
